# Docs Access

Use this reference when you need official Expo docs with low token cost.

## Best endpoints

- Site index: `https://docs.expo.dev/llms.txt`
- Full Expo bundle: `https://docs.expo.dev/llms-full.txt`
- EAS bundle: `https://docs.expo.dev/llms-eas.txt`
- Latest SDK bundle: `https://docs.expo.dev/llms-sdk.txt`

## Per-page markdown

For a single page, append `/index.md` to the docs URL.

Example:

```text
https://docs.expo.dev/develop/development-builds/create-a-build/index.md
```

## How to use this reference

- Use per-page markdown for one focused topic.
- Use `llms.txt` to discover available documentation bundles.
- Use section bundles when a task spans an Expo area such as EAS or the SDK.
- Re-check live docs when the answer depends on the latest SDK behavior, CLI flags, or rollout details.
